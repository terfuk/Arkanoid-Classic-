🧱 Arkanoid (Classic) – Multi‑Language Edition
A classic breakout / Arkanoid game where you control a paddle to bounce a ball and destroy all bricks.
Built in 7 programming languages – each implementation features a terminal‑based or simple graphical version with score tracking, lives, and increasing difficulty.

✨ Features
Paddle control – move left/right with arrow keys (or A/D).

Ball physics – bounces off walls, paddle, and bricks.

Brick layers – multiple rows of bricks with different colours (in ASCII).

Score tracking – points for each brick destroyed.

Lives system – lose a life when the ball falls off the bottom.

Level progression – after clearing all bricks, a new set appears (speed increases).

Game over – when all lives are lost.

Restart – press R to restart.

Cross‑platform – works in any terminal (or simple graphical window in some languages).

🎮 Controls
← / A – move paddle left

→ / D – move paddle right

Space – launch ball (if stuck to paddle)

R – restart after game over

Q / Esc – quit

📦 Game Mechanics
Paddle: moves horizontally at the bottom of the screen.

Ball: starts on the paddle; launch with Space.

Bricks: arranged in 3–5 rows; each hit destroys one.

Score: +10 points per brick.

Lives: start with 3; lose one when ball falls off.

Level: when all bricks are gone, a new level starts with more rows or faster ball.
